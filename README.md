## Installation
1. Clone the repository:
    ```sh
    git clone <repository-url>
    ```
2. Navigate to the project directory:
    ```sh
    cd <project-directory>
    ```
3. Install dependencies:
    ```sh
    npm install
    ```

## Usage
To start the development server:
```sh
npm start
```

## Build
To build the project:
```sh
npm run build
```
